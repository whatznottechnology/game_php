Category Filtering - How It Works

Same Page Filtering • No Reload • Instant Results

Before (Old Behavior)

1. User clicks category

onClick → Navigate to games.html?category=1
↓

2. Browser loads new page

Page reload → Slow navigation
↓

3. Shows filtered games

Separate page with filters

❌ Problems:

  • • Page reload (slow)
  • • Lost scroll position
  • • Extra HTTP request
  • • Poor UX on mobile

After (New Behavior)

1. User clicks category

onClick → filterGamesByCategory(id)
↓

2. AJAX request to API

fetch('api/games.php?category=1')
↓

3. Update games grid

No reload • Instant filtering

✅ Benefits:

  • • Instant filtering
  • • No page reload
  • • Smooth UX
  • • Mobile-friendly

Technical Flow

1

Page Load (index.html)

// When page loads
loadCategories() → Display sidebar with "All Games" + category buttons
loadFeaturedGames() → Load ALL games (not just featured)
// Initial state: All games displayed, "All Games" highlighted
2

User Clicks Category

// User clicks "Sports Betting" button
filterGamesByCategory(categoryId=1, categoryName="Sports Betting")
// Function triggered, no page navigation
3

Fetch Filtered Games

// AJAX request to backend
const response = await fetch('admin/api/games.php?category=1')
const result = await response.json()
// Backend returns only Sports Betting games
4

Update UI

// Update games grid (same page, no reload)
displayFeaturedGames(filteredGames)
// Update page title
contentTitle.textContent = "Sports Betting Games"
// Highlight active category button
activeBtn.classList.add('bg-gradient-to-r', 'from-blue-600', 'to-purple-600')
5

Result

  • Games grid updated instantly
  • No page reload
  • Category button highlighted
  • Page title updated
  • Smooth user experience

JavaScript (api.js) 

// Filter games by category
async function filterGamesByCategory(categoryId, categoryName) {
  // Update page title
  contentTitle.textContent = 
    categoryId === 'all' 
      ? 'Available Betting Platforms' 
      : categoryName + ' Games';
  
  // Highlight active category
  document.querySelectorAll('[data-category-btn]').forEach(btn => {
    btn.classList.remove('bg-gradient-to-r', 'from-blue-600', 'to-purple-600');
  });
  
  const activeBtn = document.querySelector(`[data-category-btn="${categoryId}"]`);
  activeBtn.classList.add('bg-gradient-to-r', 'from-blue-600', 'to-purple-600');
  
  // Fetch filtered games
  const url = categoryId === 'all' 
    ? 'admin/api/games.php' 
    : `admin/api/games.php?category=${categoryId}`;
    
  const response = await fetch(url);
  const result = await response.json();
  
  // Update games grid (no reload!)
  displayFeaturedGames(result.data);
}
